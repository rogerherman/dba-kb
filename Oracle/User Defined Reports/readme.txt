What Are These?
---------------
The files in this location are custom "User Defined Reports" in SQL Developer.


To Add the Report into SQL Developer:
-------------------------------------
1. Download the XML file for the report you want to use.
2. Add the report to SQL Developer.
    a. Open the "Reports" tab.
    b. Right-click the "User Defined Reports" subgroup.
    c. Select "Open Report" and choose the downloaded XML file.


Using the Report in SQL Developer
---------------------------------
Simply double click the report in the "User Defined Reports" section of the reports tab to open the report in a separate worksheet space.
